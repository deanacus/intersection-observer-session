What is Intersection Observer?

It's a browser API that allows us to asynchronously observe changes to the intersection of a given element or group of elements in relation to either the viewport or an ancestor.

It doesn't run on the main thread, so it's much more performant, the interface is super easy to use, and if does the maths for us. No more getBoundingClientRec and window.innerHeight and so forth.
